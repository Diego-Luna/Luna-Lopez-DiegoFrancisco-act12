Resolución del problema de mi comunidad

La problemática

Mi hermana y yo tenemos una comunidad y MakerSpace llamada MoonMakers, a donde buscamos que chicos y no tan chicos, conozcan más de tecnología y ciencia, junto con el movimiento Maker. También tenemos un canal de YouTube de divulgación científica y movimiento Maker. Constantemente nos preguntan si tenemos una página web para poder tener más información de nosotros y no tan sólo con nuestras redes sociales.

Así que para crear nuestra página Web necesitamos aprender Html, CSS y JavaScript. Por que al utilizar plataformas como Wix el contenido subido es de la empresa. Lo contrario sucede si nosotros la creamos, el contenido es 100% nuestro.


Recopilar datos

  1. Se necesita un Dominio.
  2. Un hosting para almacenar mi página Web.
  3. Un editor de texto.
  4. Buscar cursos a donde pueda aprender Html, Css, JavaScript.
  5. Ajustar un presupuesto.
  6. Una lluvia de ideas para ver como será la página web.

Analizar datos

  1. Ajustar un presupuesto para el hosting.
  2. Buscar en internet cursos y documentación de Html, Css y JavaScript.
  3. Comenzar a escribir en papel las ideas del sitio Web.

Descomponer problemas

  1. Descargar el editor de texto Atom .
  2. Buscar cursos de desarrollo frontend.
  3. Comenzar a hacer pruebas para la página.
  4. Crear una hoja en Excel a para anotar las diferentes ofertas de cada hosting.

Abstraer

Pasar a papel el diseño de la página Web que satisfase nuestras necesidades. Buscar documentación y tutoriales para poder hacer realidad la idea del sitio. Hacer que la página web se adapte a los tamaños de pantalla que más se usan. Y si mis conocimientos no son lo suficientes para poder realizarla, buscar en foros para ver que falta para realizarla.


Algoritmos y procedimientos

  1. Al tener la página web en mi computadora.
  2. Es momento de hacer que el diseño se adapte al tamaño de pantallas mas usadas.
  3. Comprar el hosting que va a almacenar mi sitio web.
  4. Subir mi sitio al hosting para que más personas puedan verlo.

Conclusiones

Esta unidad académica me ayudo a realizar el proyecto de una manera más eficiente, para poder ver como quedo el proyecto, aqui esta la página MoonMakers
